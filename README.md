# AWS-Terraform-Static-HTML
Automate the process of creating a static HTML page hosted on AWS

## Description
A Terraform project to automate website creation.

***Purpose of the Project***
to create S3 buckets, public bucket policies, subdomain to redirect, and upload HTML documents in an automated and repeatable process.

## Getting Started

### Dependencies
* Terraform
* AWS CLI
* An AWS account and domain already registered
* An AWS IAM user account linked to your AWS CLI and terraform instance

### Installing
Once downloaded, run the following commands
* terraform init
* terraform apply
